# SNKRS-bot
This is a SNKRS auto-purchase bot written in Python. It automates the process of buying limited-edition sneakers on Nike's SNKRS platform via their API, with built-in anti-detection features.

How it works: It logs into your Nike account, looks up a specific shoe by SKU, waits until the exact launch time, then rapidly submits purchase requests — essentially trying to beat other buyers to checkout.
```shell for running locally
pip install requests curl_cffi
```
